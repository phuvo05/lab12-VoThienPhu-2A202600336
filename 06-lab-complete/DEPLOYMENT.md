## Environment Variables Set

### Required Variables
- ✅ `PORT` - Set to 10000 (Render default)
- ✅ `REDIS_URL` - Auto-configured from Redis service
- ✅ `REDIS_ENABLED` - Set to `true`
- ✅ `API_KEY` - Generated by Render (or custom)
- ✅ `ADMIN_API_KEY` - Generated by Render (or custom)
- ✅ `RATE_LIMIT_REQUESTS` - Set to `10`
- ✅ `RATE_LIMIT_WINDOW` - Set to `60` (seconds)
- ✅ `MONTHLY_COST_LIMIT` - Set to `10.0` (dollars)
- ✅ `DEBUG` - Set to `false`

### Optional Variables
- ⚪ `OPENAI_API_KEY` - For OpenAI integration (optional)
- ⚪ `OPENAI_MODEL` - Default: `gpt-3.5-turbo`

## Deployment Steps

### Step 1: Push to GitHub
```bash
git init
git add .
git commit -m "Production AI agent - Lab 06"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
git push -u origin main
```

### Step 2: Deploy on Render (Blueprint Method)

1. Go to [Render Dashboard](https://dashboard.render.com/)
2. Click **"New"** → **"Blueprint"**
3. Connect your GitHub repository
4. Render will detect `render.yaml` and show:
   - Redis service: `ai-agent-redis`
   - Web service: `ai-agent-app`
5. Click **"Apply"**
6. Wait 3-5 minutes for deployment

### Step 3: Get Your API Keys

1. Go to your web service in Render dashboard
2. Click **"Environment"** tab
3. Find and copy:
   - `API_KEY` - For regular users
   - `ADMIN_API_KEY` - For admin access

### Step 4: Test Your Deployment

Replace `YOUR_API_KEY_HERE` with your actual API key and test:

```bash
# Quick test
curl -X POST https://your-agent.onrender.com/chat \
  -H "X-API-Key: YOUR_API_KEY_HERE" \
  -H "Content-Type: application/json" \
  -d '{"message": "Hello!"}'
```

## Troubleshooting

### Service Not Starting
1. Check logs in Render dashboard
2. Verify all environment variables are set
3. Check Redis connection status

### Redis Connection Failed
- App will fallback to in-memory storage
- Check `REDIS_URL` environment variable
- Verify Redis service is running

### Rate Limiting Not Working
```bash
# Check readiness
curl https://your-agent.onrender.com/readiness
```
If `redis: false`, check Redis connection.

### 401 Unauthorized
- Verify `X-API-Key` header is included
- Check API key is correct (copy from Render environment)
- Ensure key is at least 10 characters
